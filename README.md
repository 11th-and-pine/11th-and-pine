# 11TH-and-Pine
Capstone project documentation for 11th & Pine: a location-based audio app about CHOP.

## Project Overview
This is a capstone project that explores how the digital archive of the Capitol Hill Occupied Protest (CHOP) can be experienced through audio storytelling combined with walking through the streets where the events took place. By listening to protest narratives while moving through these locations, users are invited to reconnect with the moments when the protests unfolded. 
This project is sponsored by the University of Washington School of Drama.

---

## Project Goals
- Make CHOP-related oral histories more accessible and engaging
- Translate archival materials into an immersive and walkable experience
- Enable place-based storytelling
- Build a testable end-to-end prototype
